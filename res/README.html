STOCKS INVESTING APPLICATION

READ ME

Program Features

  • Create portfolio's (flexible or inflexible) and add or buy stocks while creating a portfolio
  • View and Composition of a portfolio on certain date
  • View the value of portfolio on full composition on a certain date
  • Buy and Add stock to the flexible portfolio
  • Sell stocks from the flexible portfolio
  • Calculate performance of the flexible portfolio on a certain date
  • Calculate the total amount invested on flexible portfolio on a certain date
  • Configure the commission cost per transaction for all the portfolio's in common
  • Add Stocks to flexible portfolio using Dollar Value strategy
  • This is a one user application. Meaning, all the date belonging to this application belongs to only one user
